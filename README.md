# 241-й день

Визуальная новелла: зомби-апокалипсис, драма, любовная линия. Полностью на русском.
Вся графика — рисованные SVG-сцены, музыка генерируется в браузере (WebAudio), файлов ресурсов нет.

## Запуск

```bash
npm start
```

Открыть http://localhost:8341 — и играть. Наушники рекомендуются.

## Управление

- **Клик / Пробел / Enter** — дальше по тексту (первый клик допечатывает строку, второй листает)
- **Выборы** — кликом; они меняют доверие и близость Ани, открывают флаги и ведут к разным концовкам
- **♪** — вкл/выкл музыку, **меню** — выход в главное меню (прогресс сохраняется автоматически)

## Концовки

4 концовки: РАССВЕТ, ОГОНЁК, ЦЕНА, ПУСТОТА. Открытые видны в галерее «Концовки».
На развилки влияют: близость (love), доверие (trust) и ключевые решения
(рассказать о прошлом, разрешить сыграть на пианино, обещание вернуться,
рвануть сумку или бросить, сказать про укус или скрыть, север или заправка).

## Тесты

```bash
npm test          # Playwright e2e (13 тестов)
```

Тесты проверяют меню, целостность графа сюжета (все переходы ведут на существующие узлы),
влияние выборов на статы, сохранение/продолжение и достижимость всех четырёх концовок.
Режим `/?fast=1` отключает анимацию печати текста для тестов.

## Структура

- `js/story1.js`, `js/story2.js` — сюжет (~110 узлов): пролог + 5 глав + концовки
- `js/engine.js` — движок: печать текста, выборы, статы, сохранения, частицы (дождь/пепел/снег/искры)
- `js/backgrounds.js` — 12 SVG-сцен и портреты персонажей
- `js/audio.js` — процедурный эмбиент: 5 тем с кроссфейдом
